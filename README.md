# Kendoplan App

Static training schedule site for GitHub and Vercel.

## Files

- `index.html`: published page
- `training_schedule.json`: source of the schedule data

## Update flow

1. Edit `training_schedule.json`
2. Commit and push to GitHub
3. Vercel redeploys automatically

## Deploy

Import this repository into Vercel and deploy it as a static site without a build command.
